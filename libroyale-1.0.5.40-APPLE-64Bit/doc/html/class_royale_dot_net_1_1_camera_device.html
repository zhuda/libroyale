Detailed Description

This class represents the time-of-flight camera system.

Typically, an instance is created by the CameraManager which automatically detects a connected module. The support access levels can be activated by entering the correct code during creation. After creation, the CameraDevice is in ready state and can be initialized. Before starting capturing an operation mode must be set. The provided examples should give a good overview on how to use this class.
