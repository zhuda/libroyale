Detailed Description

template<class Category, class T, class Distance = ptrdiff_t, class Pointer = T *, class Reference = T &>
class royale::royale_reverse_iterator< Category, T, Distance, Pointer, Reference >

Royale non-const reverse iterator class (used by royale containers)

Holding the specific functions for the non-const reverse iterator. Other specific iterators shall not be based on this final iterator anymore.

The non-const reverse iterator is used by the royale container classes and is compatible with the rest of the STL implementation concerning iterators
