virtual void royale::IDepthDataReleaser::releaseDepthData ( const DepthData *  data)
pure virtual

Must be called on all DepthData pointers that are provided by the IDepthDataListener interface.

This returns buffer ownership to the processing module, and allows the memory to be reused.

This should not be called when using the Royale API, as CameraDevice releases the depth data when that IDepthDataListener callback returns.
