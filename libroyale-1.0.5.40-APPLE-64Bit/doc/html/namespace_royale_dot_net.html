enum RoyaleDotNet.CallbackData
strong

Specifies the type of data which should be captured and returned as callback.

Enumerator
None 

only get the callback but no data delivery

Raw 

raw frames, if exclusively used no processing pipe is executed (no calibration data is needed)

Depth 

one depth and grayscale image will be delivered for the complete sequence

Intermediate 

all intermediate data will be delivered which are generated in the processing pipeline

enum RoyaleDotNet.OperationMode
strong

Each camera can be operated in various operation modes.

This list is a complete list of modes which are supported by Royale. However, every camera device only allows a sub-set of this list. The ICameraDevice can be asked about the supported operation modes by calling getOperationModes(). This will return a list of supported modes while the first one will be the default mode.
