Detailed Description

template<class Category, class T, class Distance, class Pointer, class Reference>
class royale::royale_iterator_skeleton< Category, T, Distance, Pointer, Reference >

Iterator skeleton.

Holding the basic functions which are shared between all types of royale iterators. Handling virtual functions for incrementing and decrementing pointers. This is the very basic class for handling iterators; based on the STL template definitions which makes it STL compatible - all royale iterators are based in this skeleton and every upcoming iterators will also be based on this skeleton.
